I/O Chainrings

I built my own personal set of custom Track Rings. In between each bolt-hole, there are 6 cutouts. These 6 cutouts represent 6 bits.

Ones are represented by holes, zeros are represented by the partial cutouts. Reading from left to right, the 6 cutouts represent the binary code value for the number of teeth in the chainring.

Design based on AARN Chainrings Designed by Aaron Panone.
